## 1. 概要

本仕様書は、香港地下鉄 TCL線ソフトについて、香港地下鉄 MTRC線からの変更点のみを記す。

---

## 2. 構成

### 2.1. ハードウェア構成

MTRC線から構成に変更なし。  
ただし、TMSシミュレータボードが、旧DVAU基板がもう無いために、新DVAU基板での対応となり、一部ソフトを変更する必要がある。

### 2.2. ソフトウェア構成

MTRC線から変更無し。

---

## 3. ソフト変更内容

### 3.1. DVAU1

- **ソフトバージョン**  
  MTRCの Ver1.XX に対し、TCL は Ver.3.XX とする。  
  また、変更元 MTRC は Ver1.10 である。

- **MCPへの非常ハンドルの表示文字変更**  
  `'ＰＥＡＨ'` → `'ＰＡＤ'` へ変更。

- **非常ハンドルの物理位置と表示位置の変換**  
  MTRCと車両の連結方向が一部異なるため、非常ハンドルの No. を逆順に処理する必要がある。  
  これは表示のみで、データ自体は MTRC と同等に行う。  
  ＰＡＡ アドレス２、４、６、７は逆順に表示する処理を追加。

- **CarNo. の変更**

  | 旧 | 新 |
  |----|----|
  | A1 A2 A3 A4 A5 | V1 V2 V3 V4 V5 |
  | C1 C2 C3 C4 C5 | Z1 Z2 Z3 Z4 Z5 |
  | B1 B2 B3 B4 B5 | X1 X2 X3 X4 X5 |
  | B1-1 B1-2 B1-3 B1-4 B1-5 | W1 W2 W3 W4 W5 |
  | C1-１ C1-2 C1-3 C1-4 C1-5 | Y1 Y2 Y3 Y4 Y5 |
  | B1 B2 B2 B2 B2 | X1 X2 X3 X4 X5 |
  | C1 C2 C2 C2 C2 | Z1 Z2 Z3 Z4 Z5 |
  | A1 A2 A2 A2 A2 | V1 V2 V3 V4 V5 |

---

### 3.2. TMSシミュレータ

- **PCカードの対応**  
  PCカードのクラスタ／セクター数を **16 → 32** へ変更。

- **メモリマップの変更**

  | 項目 | 変更前 | 変更後 |
  |------|--------|--------|
  | RAM 先頭番地 | 0x100000 | 0x080000 |
  | RAM サイズ | 0x080000 | 0x020000 |
  | Boot イメージサイズ | 0x060000 | 0x015000 |

- **CarNo. の変更**

  | 旧 | 新 |
  |----|----|
  | A1 A2 A3 A4 A5 | V1 V2 V3 V4 V5 |
  | C1 C2 C3 C4 C5 | Z1 Z2 Z3 Z4 Z5 |

  | B1-1 B1-2 B1-3 B1-4 B1-5 | W1 W2 W3 W4 W5 |
  | C1-１ C1-2 C1-3 C1-4 C1-5 | Y1 Y2 Y3 Y4 Y5 |
  | B1 B2 B2 B2 B2 | X1 X2 X3 X4 X5 |
